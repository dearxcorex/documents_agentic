# หนังสือภายใน / บันทึกข้อความ (Internal Memo Template)

> Organization: สำนักงาน กสทช. (NBTC)

## Structure

```
┌─────────────────────────────────────────────────────────────────────────┐
│                            บันทึกข้อความ                                 │
├─────────────────────────────────────────────────────────────────────────┤
│ หน่วยงาน  [สำนักงาน/เขต] ([งาน/ฝ่าย] โทร. [เบอร์ภายใน])                   │
│ ที่  สทช [รหัสหน่วยงาน]/              วันที่  [วัน เดือน พ.ศ.]            │
│ เรื่อง  [หัวข้อเรื่อง]                                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│ เรียน  [ตำแหน่งผู้รับ]                                                   │
│                                                                         │
│ ๑. เรื่องเดิม                                                           │
│                                                                         │
│     1.1 [อ้างอิงบันทึก/หนังสือ ที่เกี่ยวข้อง]                              │
│     1.2 [อ้างอิงเพิ่มเติม (ถ้ามี)]                                        │
│                                                                         │
│ ๒. เรื่องเพื่อพิจารณา                                                    │
│                                                                         │
│     2.1 [รายละเอียดที่ขอพิจารณา]                                         │
│         2.1.1 [รายการย่อย]                                               │
│         2.1.2 [รายการย่อย]                                               │
│                                                                         │
│         จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ                                  │
│                                                                         │
│                                              [ลายมือชื่อ]                 │
│                                             ([ชื่อ-นามสกุล])              │
│                                             [ตำแหน่ง]                    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## Document Number Format (รหัสหน่วยงาน)

| รหัส | หน่วยงาน |
|------|----------|
| สทช 2203.3 | สำนักงาน กสทช.เขต 23 - งานตรวจสอบและกำกับดูแล |
| สทช 2401 | กองตรวจสอบและกำกับดูแลกิจการกระจายเสียง |

---

## Numbering Convention

| Level | Format | Example |
|-------|--------|---------|
| Section | Thai numeral | ๑. ๒. ๓. |
| Sub-section | Arabic decimal | 1.1 1.2 2.1 |
| Item | Arabic sub-decimal | 2.1.1 2.1.2 2.1.3 |

## Example: Travel Approval Request (ขออนุมัติเดินทาง)

```
                            บันทึกข้อความ

หน่วยงาน  สำนักงาน กสทช.เขต 23 (งานตรวจสอบและกำกับดูแล โทร. 4413)
ที่  สทช 2203.3/                       วันที่  20 มกราคม 2569
เรื่อง  ขออนุมัติเดินทางไปปฏิบัติงานนอกที่ตั้ง

เรียน  ผภภ. 23

๑. เรื่องเดิม

    1.1 ตามบันทึกกองตรวจสอบและกำกับดูแลกิจการกระจายเสียง ที่ สทช.
        2401/123 ลงวันที่ 15 มกราคม 2569 เรื่อง ขอความร่วมมือตรวจสอบ
        สถานีวิทยุ รายละเอียดตามเอกสารแนบ 1 นั้น

    1.2 [ในกรณีที่มีหลายงาน - เพิ่มรายการอ้างอิงที่นี่]

๒. เรื่องเพื่อพิจารณา

    2.1 เพื่อให้การปฏิบัติงานตามข้อ ๑. เป็นไปด้วยความเรียบร้อย
        จึงขออนุมัติให้พนักงาน ภภ. 23 เดินทางไปปฏิบัติหน้าที่ตรวจสอบ
        สถานีวิทยุชุมชน ระหว่างวันที่ 25-27 มกราคม 2569
        ในพื้นที่จังหวัดเชียงใหม่ และพื้นที่ใกล้เคียง โดยมีพนักงานดังนี้

        2.1.1 นายสมชาย ใจดี       ตำแหน่ง พนักงานปฏิบัติการระดับกลาง
        2.1.2 นายสมศักดิ์ รักงาน   ตำแหน่ง พนักงานปฏิบัติการระดับต้น
        2.1.3 นางสาวสมหญิง ขยัน  ตำแหน่ง พนักงานปฏิบัติการระดับต้น

        จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ

                                              (นายประสิทธิ์ ผลงาน)
                                              ผู้อำนวยการสำนัก กสทช. เขต 23
```

## Example: Report/Notification (รายงานผล)

```
                            บันทึกข้อความ

หน่วยงาน  สำนักงาน กสทช.เขต 23 (งานตรวจสอบและกำกับดูแล โทร. 4413)
ที่  สทช 2203.3/                       วันที่  30 มกราคม 2569
เรื่อง  รายงานผลการตรวจสอบสถานีวิทยุชุมชน

เรียน  ผภภ. 23

๑. เรื่องเดิม

    1.1 ตามบันทึก สทช 2203.3/xxx ลงวันที่ 20 มกราคม 2569 เรื่อง
        ขออนุมัติเดินทางไปปฏิบัติงานนอกที่ตั้ง ซึ่งได้รับอนุมัติแล้วนั้น

๒. ผลการดำเนินงาน

    2.1 พนักงานได้เดินทางไปปฏิบัติหน้าที่ตรวจสอบสถานีวิทยุชุมชน
        ระหว่างวันที่ 25-27 มกราคม 2569 ในพื้นที่จังหวัดเชียงใหม่
        ผลการตรวจสอบสรุปได้ดังนี้

        2.1.1 สถานีที่ตรวจสอบ จำนวน 15 สถานี
        2.1.2 สถานีที่ดำเนินการถูกต้อง จำนวน 12 สถานี
        2.1.3 สถานีที่ต้องปรับปรุง จำนวน 3 สถานี

    2.2 รายละเอียดตามเอกสารแนบ

        จึงเรียนมาเพื่อโปรดทราบ

                                              (นายสมชาย ใจดี)
                                              พนักงานปฏิบัติการระดับกลาง
```

## JSON Template

```json
{
  "template_type": "internal",
  "organization": "กสทช",
  "sections": {
    "header": {
      "title": "บันทึกข้อความ",
      "agency": {
        "name": "string (e.g., สำนักงาน กสทช.เขต 23)",
        "division": "string (e.g., งานตรวจสอบและกำกับดูแล)",
        "phone": "string (internal extension)"
      },
      "document_number": {
        "prefix": "สทช",
        "code": "string (e.g., 2203.3)",
        "sequence": "string (optional running number)"
      },
      "date": "string (Thai format: วัน เดือน พ.ศ.)"
    },
    "subject": "string",
    "recipient": {
      "title": "เรียน",
      "position": "string (e.g., ผภภ. 23)"
    },
    "body": {
      "sections": [
        {
          "number": "๑",
          "title": "เรื่องเดิม",
          "items": [
            {
              "number": "1.1",
              "content": "string (reference to documents)",
              "reference": {
                "agency": "string",
                "document_number": "string",
                "date": "string",
                "subject": "string",
                "attachment": "number (optional)"
              }
            }
          ]
        },
        {
          "number": "๒",
          "title": "เรื่องเพื่อพิจารณา | ผลการดำเนินงาน",
          "items": [
            {
              "number": "2.1",
              "content": "string",
              "sub_items": [
                {
                  "number": "2.1.1",
                  "content": "string"
                }
              ]
            }
          ]
        }
      ]
    },
    "closing": {
      "phrase": "string",
      "signature": {
        "name": "string",
        "position": "string"
      }
    }
  }
}
```

## Common Section Titles

| Section ๒ Title | Use Case |
|-----------------|----------|
| เรื่องเพื่อพิจารณา | Request for approval |
| ผลการดำเนินงาน | Report results |
| ข้อเสนอ | Proposal |
| ความเห็น | Opinion/Recommendation |

## Common Closing Phrases

| Purpose | Closing Phrase |
|---------|----------------|
| For approval | จึงเรียนมาเพื่อโปรดพิจารณาอนุมัติ |
| For information | จึงเรียนมาเพื่อโปรดทราบ |
| For consideration | จึงเรียนมาเพื่อโปรดพิจารณา |
| For action | จึงเรียนมาเพื่อโปรดดำเนินการ |
| For acknowledgment | จึงเรียนมาเพื่อโปรดทราบและถือปฏิบัติ |

## Reference Pattern (รูปแบบการอ้างอิง)

```
ตามบันทึก [ชื่อหน่วยงาน] ที่ สทช. [เลขที่หนังสือ] ลงวันที่ [วันที่]
เรื่อง [ชื่อเรื่อง] รายละเอียดตามเอกสารแนบ [หมายเลข] นั้น
```
